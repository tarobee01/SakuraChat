<h1 style="font-size: 2em;">SakuraChat</h1>
日本語学習者を目的とした、ソーシャル掲示板アプリ。
<br></br>
<br></br>

<h1 style="font-size: 1.5em;">使用した技術</h2>
・Swift, SwiftUI<br></br>
・naturalLanguage, combine, photosUI<br></br> 
・firebase<br></br>
・github, xcode<br></br>
